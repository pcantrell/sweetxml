SweetXML README

This directory contains the source distribution of SweetXML, in the following modules:

  • sweetxml-core: The core classes, including SweetXML ↔ XML converters and the command-line interface. Different languages will live under src/java, src/ruby, src/python, etc.
  • sweetxml-ant: The SweetXML Ant task.
  • sweetxml-maven: The SweetXML Maven plugin.

Building the Java code requires Maven.

For more information, see the SweetXML home page.
