SweetXML README

This directory contains the source distribution of SweetXML, in the following modules:

  • core-java: The core Java classes, including SweetXML ↔ XML converters and the command-line interface. Different languages will live under core-ruby, core-python, etc.
  • ant: The SweetXML Ant task (not yet started).
  • maven: The SweetXML Maven plugin (not yet complete).
  • housekeeping: Scripts for maintaining the project.

Building the Java code requires Maven.

For more information, see the SweetXML home page.
