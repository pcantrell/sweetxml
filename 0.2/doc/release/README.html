SweetXML ${version}

SweetXML is an alternate syntax for XML, designed to make configuration files more concise and readable by adding a bit of syntactic sugar.

This distribution includes the following:

sweetxml-${version}.jar core library
sweetxml-ant-${version}.jar Ant plugin
sweetxml-maven-${version}.jar Maven plugin
bin/ command-line interface and helper scripts
doc/ documentation

For help getting started, read the user guide.

Questions or suggestions? Feel free to share them.

What's New in ${version}

  • Implemented Ant task.
  • Implemented Maven plugin.
  • XML to SweetXML conversion now preserves more user formatting, and handles inline DOCTYPEs.
  • Blackslash escape syntax removed in favor of normal XML escaping.
  • Conversion regression testing.

For complete version history and future plans, see the roadmap.
